# About
I always wanted to have a few of weather forecast in one place to compare them to each other. Obviously goal of this project to end up with exactly that hopefuly with usable interface (at least text-based).
Not sure how useful it will end up being as the number of free weather apis is limited.
Currently working with Yr and Open-Meteo. Eventualy would be nice to include Meteoblue api as it has some free yearly usage.
Goal for now is to implement core logic for api integration and data manipulation.
